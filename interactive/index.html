BINARY SEARCH

Apa itu Binary Search?

Binary search adalah algoritma pencarian yang efisien untuk menemukan posisi nilai target dalam array yang sudah diurutkan. Algoritma ini bekerja dengan membagi interval pencarian menjadi dua bagian pada setiap langkahnya, sehingga kompleksitas waktunya adalah O(log n).
