Mathematics Visualization - Graph Function

Explore and visualize mathematical functions by entering a formula. Understand graph shapes for various mathematical functions such as linear, quadratic, and exponential.

Function Examples
  • Linear: y = ax + b
  • Quadratic: y = ax^2 + bx + c
  • Cubic: y = ax^3 + bx^2 + cx + d
  • Exponential: y = a^x
  • Logarithmic: y = log(x)
Variable Definitions
  • a, b, c, d: Coefficients
  • x: Variable (independent)
  • +: Addition
  • ^: Exponentiation
